# Wer wird Millionär? - Terminal-Spiel

Willkommen zur Terminal-Version von "Wer wird Millionär?"! Dieses Spiel ermöglicht es Ihnen, Ihr Wissen zu testen und virtuelles Geld zu gewinnen, indem Sie eine Reihe von Fragen beantworten.

## So spielen Sie

1. **Spiel starten**: Führen Sie das Spielskript aus, um das Spiel zu starten.
2. **Fragen beantworten**: Ihnen werden eine Reihe von Fragen mit steigenden Gewinnsummen gestellt.
3. **Joker verwenden**: Sie haben zwei Joker zur Verfügung:
   - **50-50 Joker**: Entfernt zwei falsche Antworten.
   - **Publikumsjoker**: Zeigt die Abstimmungsprozentsätze des Publikums für jede Option an.
4. **Sicherheitsnetze**: Es gibt zwei Sicherheitsnetze bei 1.000 $ und 32.000 $. Wenn Sie eine Frage falsch beantworten, nehmen Sie den Betrag des letzten Sicherheitsnetzes mit nach Hause.
5. **Jederzeit aufhören**: Sie können jederzeit aufhören und Ihren aktuellen Gewinn mit nach Hause nehmen.

## Spielregeln

1. Es gibt 15 Fragen mit steigenden Gewinnsummen.
2. Sie haben zwei Sicherheitsnetze: bei 1.000 $ und 32.000 $.
3. Sie können jederzeit aufhören und Ihren aktuellen Gewinn mit nach Hause nehmen.
4. Sie haben zwei Joker: 50-50 und Publikumsjoker.
5. Viel Glück!

## Spiel ausführen

1. Stellen Sie sicher, dass Node.js auf Ihrem Rechner installiert ist.
2. Installieren Sie die `readline-sync` Bibliothek, falls noch nicht geschehen:
   ```bash
   npm install readline-sync
